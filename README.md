## Pre-requisites
- NPM.
- Node.js.

## Setting Up
1. Install Gatsby: ```npm install --global gatsby-cli```
2. Install modules: ```npm install```.
3. Start the website by running: ```gatsby develop```.
4. The site is now running at http://localhost:8000.
5. Stop the site with cntrl + c.
